# OmniAgent 真实页面 E2E 执行记录模板

本目录只保存脱敏后的测试报告与证据索引。不得提交 Cookie、令牌、账号密码、真实私密文档、SSH 私钥或未脱敏控制台日志。

## 文件命名

- 报告：`OMNI-E2E-YYYYMMDD-NN.md`
- 证据目录：`OMNI-E2E-YYYYMMDD-NN/`
- 截图：`<用例ID>-<步骤>-<结果>.png`
- 日志：`<用例ID>-console-sanitized.txt`

## 执行规则

1. 每个用例执行前记录初始数据库数量；
2. 页面显示成功时必须同步核对持久化记录；
3. 跨平台召回必须记录来源 Evidence 和作用域；
4. 失败必须保留脱敏日志和最短复现步骤；
5. 修复必须关联 Commit 与新增自动回归测试；
6. 一轮报告只对应一个构建，不得中途替换扩展代码；
7. P0/P1 未清零时，结论不得填写“通过”。
